Online Detection

Colors:

  • Green indicates the device was online.
  • Gray indicates the device is offline.
  • Pink indicates the network probe was down or its a future time.

Clicking on a online/offline device bar will open a dialog where you can edit the device information.

When a device connects to the network, it is identified by its MAC address. This address is unique to each network interface card (NIC) and can be used to track the device's online status.

A computer can show up multiple times with different MAC addresses because it can have multiple NICs. For example, one for Ethernet and one for WiFi. They generally not appear online simultaneously.

Most cellphones randomize their Machine Address (MAC) to help prevent user tracking. They may use the same MAC when reconnecting to the same network.

Select List Administration

The Codes/Descriptions are used in the select lists for users to make a choice, such as the names of the offices.

  • Code - Value stored in the database.
  • Description - What the user selects onscreen.
  • Active - Sets if it still can be used.

To set the display order in the select list, use the Up/Down Arrows.

When the code is not used in any records, a red X will allow you to remove it.

The Asset ID prefixes are prepended on the Asset ID when creating a device record.

These prefixes can be used to help identify the type of the device

For example, if the prefix is "LAPTOP", the Asset ID will be "LAPTOP-123456"

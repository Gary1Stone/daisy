DAISY

Device Asset Information SYstem

An environment for recording the electronic assets of a small non-profit organization. All intellectual property rights, copyrights and photos are retained by the author. This product was designed and developed for:

WKNC

Weston King Neighbourhood Centre

DAISY is constructed using the following components:

  • Metro 4 an open-source user interface framework. Many thanks to Pimenov Serhii. Buy him a coffee
  • SQLite an open source public domain database. Version 3.35.5
  • Go Programming Language An open-source programming language released under Creative Commons license.
  • MD5 encryption provided by Sebastian Tschan released under the MIT license.
  • Justin Nguyen provided a great article on solving the law of cosines theorem with SQLite.
  • Simplemaps provided world cities geographic information under a Creative Commons license.
  • Opendatasoft provided geographic information for world communities listing with a population over 1000.

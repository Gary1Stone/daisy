To look for icons:
https://tabler.io/icons

When you find one, COPY it from icons_ALL into icons

Link to it with: <img src="/icons/user.svg" alt="user icon" />

OR

Read the file to insert it where you want, then you can change the size/color with CSS
<a href="#">
  <svg xmlns="http://www.w3.org/2000/svg" class="icon" width="24" height="24" viewBox="0 0 24 24" stroke="currentColor" fill="none">
    <!-- SVG paths here -->
  </svg>
  Example
</a>
